`key`
----------

**Type:** `string`

**Value:** `scp | ssh`

**Attribute:** required

**Parent:** None

Type of command, support standard `scp` and `ssh` now.  

- If set as `scp`, then [`value`](value.md) should be [`scp_cmds`](scp_cmds.md) only

- If set as `ssh`, then [`value`](value.md) should be [`ssh_cmds`](ssh_cmds.md) only
