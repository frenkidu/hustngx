`http_basic_auth_file`
----------

**Type:** `string`

**Value:** `<file-url>`

**Example:** `"http_basic_auth_file": "/data/hustmqha/conf/htpasswd"`

**Parent:** None

Path of key file used by `http_basic_auth_module`. The username and password are stored as plain text in key file, for example:  

    jobs:p@ssword
