main_conf
----------

**Type:** `array`

**Value:** `[<conf>, <conf>, ..., <conf>]`

**Attribute:** Optional

**Parent:** None

Global configuration items for nginx module. Each item is a [conf](conf.md).  

**If the field is ignored, code on configuration of nginx will not be generated**.
