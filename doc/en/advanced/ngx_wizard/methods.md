`methods`
----------

**Type:** `array`

**Value:** `[<method>, <method>, ... , <method>]`

**Attribute:** Optional

**Parent:** [handler](handler.md)

Supported http methods for `handler` in nginx module. Each item is a [method](method.md).  

**If this field is ignored, no `http method` will be filtered**.
