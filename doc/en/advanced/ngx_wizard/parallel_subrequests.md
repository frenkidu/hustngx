`parallel_subrequests`
----------

**Type:** `bool`

**Value:** `true | false`

**Attribute:** Optional

**Parent:** [upstream](upstream.md)

If this field is used, `hustngx` will generate code template on `parallel subrequests`. See more details from [here](../parallel_subrequests.md).
