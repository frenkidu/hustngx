`path`
----------

**Type:** `object`

**Value:** `["<key>": <value>, "<key>": <value>, ..., "<key>": <value>]`

**Attribute:** Required

**Parent:** [sync_file](sync_file.md)

Path information for sync. You can set string of `key` as below:  

- [`prefix`](prefix.md)

- [`files`](files.md)

- [`suffix`](suffix.md)
